A aplicação é uma console application e foi desenvolvida utilizando .NET Core 2.0 
Escrevi o programa utilizando inglês como padrão de nomenclatura para facilitar o entendimento do código em qualquer país.

Ao executar o sistema, é necessário passar o caminho do log como parametro do executável.
